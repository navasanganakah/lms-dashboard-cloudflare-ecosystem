# ns_lms_os_sdk

**A Professional Flutter SDK for Cloudflare-backed LMS Integration.**

Created and maintained by **NavaSanganakah Multiventures**.
**Lead Developer:** Acharya Pandit Dheerendra Tripathi

## Overview

`ns_lms_os_sdk` is an open-source Flutter SDK designed to seamlessly connect your Flutter applications with a Cloudflare Workers and D1 backend. It provides both headless data layers (for custom UI) and plug-and-play UI widgets.

## Features

- **Headless Data Layer:** Direct access to APIs without being forced to use specific UI.
- **Plug-and-Play UI:** Ready-to-use widgets like `NsLoginScreen`, `NsCourseList`, and `NsChatView`.
- **Real-Time Chat:** Built-in WebSocket management for Cloudflare Durable Objects.
- **Secure Authentication:** Email + OTP login flow with secure session storage.
